# Анализ сетевых угроз с использованием Python

## Описание проекта

Данный проект представляет собой простой инструмент анализа сетевых угроз, реализованный на языке Python.

Программа анализирует события сетевой безопасности из логов **Suricata**, при необходимости проверяет подозрительные IP-адреса через **VirusTotal API**, определяет потенциальные угрозы и формирует отчёт с визуализацией результатов.

Проект реализует полный цикл обработки данных:

* сбор данных
* анализ
* реагирование на угрозы
* формирование отчёта

# Этапы выполнения задания

## Этап 1. Сбор данных

Используются два источника данных:

* **логи Suricata** — события сетевой безопасности
* **API VirusTotal** — проверка подозрительных IP-адресов

Программа загружает события из JSON-логов Suricata и извлекает IP-адреса источников сетевой активности.

Если задан API-ключ VirusTotal (через переменную окружения), программа дополнительно выполняет проверку IP-адресов через API.

Если API-ключ отсутствует — анализ выполняется только на основе логов.

---

## Этап 2. Анализ данных

Скрипт выполняет следующие действия:

* загружает логи Suricata
* извлекает IP-адреса источников событий
* подсчитывает количество событий для каждого IP
* определяет наиболее активные IP-адреса
* при наличии API-ключа выполняет проверку через VirusTotal

Для анализа используются библиотеки:

* pandas
* json
* requests

Подозрительными считаются IP-адреса, если:

* количество событий превышает заданный порог
* или VirusTotal сообщает о вредоносной активности

---

## Этап 3. Реагирование на угрозы

Если обнаружены подозрительные IP-адреса, выполняется реагирование.

В текущей версии реализована **имитация блокировки IP-адреса** — программа выводит сообщение в консоль.

Пример:

```
Обнаружены подозрительные IP-адреса. Запуск реагирования...
[РЕАКЦИЯ] Блокировка IP-адреса: 84.24.39.95
```

Модуль реагирования вынесен в отдельный файл и может быть расширен для:

* блокировки через firewall
* iptables
* сетевые устройства
* отправки уведомлений (Telegram, email)

---

## Этап 4. Формирование отчёта и визуализация

После анализа программа формирует отчёт:

* **CSV отчёт**
* **JSON отчёт**

Также строится график:

**Топ-5 IP-адресов по количеству событий**

График сохраняется в формате:

```
PNG
```

```
pip install requests pandas matplotlib seaborn
```

---

# Переменная окружения VirusTotal (опционально)

Для проверки IP через VirusTotal необходимо задать API-ключ.

Linux / macOS:

```
export VT_API_KEY=your_api_key
```

Windows:

```
set VT_API_KEY=your_api_key
```

Если ключ не задан — анализ выполняется только по логам Suricata.
